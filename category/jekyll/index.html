Jekyll环境配置

2017年10月17日 Jekyll

1. 安装RVM

安装RVM:
$ curl -L get.rvm.in | bash -s stable
指定源:
$ source ~/.rvm/scripts/rvm
查看版本:
$ rvm -v

2. 安装Ruby

查看已知版本:
$ rvm list known
安装Ruby:
$ rvm install 2.4.2
