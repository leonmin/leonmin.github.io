Git push false

leonmin
2018年4月14日

这个博客是由另一个博客copy过来，但修改样式就弄了一天，昨晚还弄到了3点，然后git push到远程仓库,结果git push失败了，这几次都失败了，每次都删了库重新git push，很麻烦，但实在太困了就去睡了。

早上11点左右起来，继续弄git push，怎么弄就是不对，反反复复把本地代码整没了。哇，顿时心态爆炸，只有重新copy来重新修改样式。

以前也会写东西或做项目忘记保存，突然死机什么的可能会丢失工作。但在我的常规思维里Git还是比较安全的，一方面代码已经托管在网上了，相当于备了份，另一方面，还可以回滚到以前的版本。结果，没想到再安全，通过自己的瞎搞都会变得不安全。

吃一堑，长一智。有时间还是好好研究一下Git。

今天又花了一天的时间做了昨天做过的事。
